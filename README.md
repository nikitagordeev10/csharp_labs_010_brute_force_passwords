#### Практика «Перебор паролей 2»
##### Решено 05 ноября 2022 в 21:35

Программа, которая по заданному слову перебирает все возможные пароли, полученные из этого слова заменой регистра.

Уже создан проект, который будет тестировать программу. Я дописал класс CaseAlternatorTask в одноименном файле.

Для удобства пароли появляются в лексикографическом порядке, считая, что маленькие буквы меньше больших. 

Например, для входного слова 'ab42' результат должен быть такой: 'ab42', 'aB42', 'Ab42', 'AB42'

На вход подается слово в нижнем регистре. В результирующем списке нет быть повторений слов.

У нас в распоряжении есть методы char.IsLetter, char.ToLower и char.ToUpper.
